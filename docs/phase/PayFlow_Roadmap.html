PayFlow - Development Roadmap

Nguyên tắc: Mỗi phase đều chạy được, demo được và xây dựng trên phase trước.

Phase 0 - Infrastructure

  • Docker Compose
  • PostgreSQL
  • Kafka + Kafka UI
  • Redis
  • Spring Cloud Gateway
  • CI cơ bản
Demo: docker compose up → toàn bộ hạ tầng hoạt động.

Phase 1 - Authentication

  • User Service
  • Register/Login
  • JWT + Refresh Token
  • Role USER/ADMIN
  • React Login/Register

Phase 2 - Wallet

  • Wallet Service
  • Tạo ví
  • Xem số dư
  • Ledger là bảng trong Wallet
  • Dashboard hiển thị Balance

Phase 3 - Top-up

  • Payment Gateway Fake
  • Kafka Event-driven
  • Notification Service
  • Outbox Pattern
  • Idempotency Key
  • Retry publish Kafka
  • DLQ Notification
  • Audit Top-up

Phase 4 ⭐ - Transfer Money

  • Transaction Service
  • Saga Pattern
  • Optimistic Lock
  • Rollback
  • Ledger
  • OpenTelemetry + Zipkin
  • Prometheus Metrics

Phase 5 - Merchant

  • Merchant Registration
  • Merchant Approval
  • Merchant Dashboard
  • Role-based Authorization

Phase 6 ⭐⭐⭐ - QR Payment

  • Generate QR
  • Scan QR
  • Payment cho Merchant
  • Tái sử dụng Saga
  • Circuit Breaker

Phase 7 - Real-time

  • SSE
  • Merchant Dashboard Realtime
  • Prometheus
  • Grafana

Phase 8 - Refactor

  • Tách Ledger thành Ledger Service
  • Chuẩn bị Audit, Reporting, Reconciliation

Phase 9 - Settlement

  • Settlement Service
  • Merchant Withdraw
  • Batch Settlement
  • Retry + Circuit Breaker

Phase 10 - Hardening

  • OAuth2 / Spring Authorization Server hoặc Keycloak
  • Rate Limiting
  • Audit
  • Alert Rule
  • Integration Test (Testcontainers)
  • Load Test (k6/JMeter)
